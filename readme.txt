Como ejecutar

Reemplazar en /Backend/server.js las constantes user, password, y database por las equivalentes de la tabla de MySQL que usa. La tabla de SQL debe tener columnas llamadas id, name, email, y password.

Ejecutar con node desde la terminal (npm start) la carpeta Backend
Ejecutar con node desde la terminal (npm run dev) la carpeta frontEnd

La tabla con la información de la database debería estar en http://localhost:5173/
